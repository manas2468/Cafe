# The Limelight Cafe Assets Folder

Drop your own images and videos into this directory (`public/limelight/`) to automatically replace the placeholders on the website. No code changes or rebuild steps are required!

### 🎥 Video Assets
- **`hero-bg.mp4`**: Drop your background video loop here to replace the default hero background video.

### 🖼️ Cafe & Food Images
- **`story-bg.jpg`**: Main image showcasing the vibe/interior of the cafe in the "Our Heritage" section.
- **`menu-item-1.jpg`**: Image for "Paneer Tikka Cheese Burst Pizza".
- **`menu-item-2.jpg`**: Image for "Brown Sugar Shaken Espresso".
- **`menu-item-3.jpg`**: Image for "Hazelnut Frappé".
- **`pairing-1.jpg`**: First food pairing image (Tandoori Paneer Roll).
- **`pairing-2.jpg`**: Second food pairing image (Cheese Garlic Bread).

### 📸 Instagram Feed Mock Grid
- **`ig-1.jpg`** to **`ig-6.jpg`**: Snaps from your Instagram feed to render in the mock social grid at the bottom.

*Note: Make sure the file names match exactly and use lowercase extension formats (like `.mp4` or `.jpg`).*
